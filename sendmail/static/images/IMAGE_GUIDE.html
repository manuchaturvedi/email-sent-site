6How to Add Images

  1. Download or create your images according to the specifications above
  2. Rename them exactly as shown (case-sensitive):
    • logo.png
    • hero-bg.jpg
    • profile-icon.jpg
    • automation-icon.jpg
    • jobs-icon.jpg
    • email-icon.jpg
  3. Copy the images to: c:\Users\windows 10\Desktop\AI_support\sendmail\static\images\
  4. Restart your Flask application or refresh your Docker container
  5. Visit your site - the images will automatically appear!

7Image Optimization Tips

  • Keep file sizes under 500KB for fast loading
  • Use JPG for photos (hero backgrounds, feature images)
  • Use PNG for logos (supports transparency)
  • Compress images using TinyPNG or Squoosh
  • Use descriptive filenames for SEO

8Fallback Behavior

üìå Important: If an image is missing, the site will gracefully fall back to:
