# How to Add Your Logo and Images

## Step 1: Add Your Logo
1. Save your logo image as `logo.png` (or `logo.jpg`, `logo.svg`)
2. Place it in: `sendmail/static/images/logo.png`
3. Recommended size: 200x50px (transparent background preferred)

## Step 2: Add Hero Poster/Banner
1. Save your hero image as `hero-bg.jpg`
2. Place it in: `sendmail/static/images/hero-bg.jpg`
3. Recommended size: 1920x600px (landscape)

## Step 3: Add Feature Images
Add these images to `sendmail/static/images/`:
- `dashboard-icon.png` - Dashboard overview image
- `profile-icon.png` - Profile setup image
- `automation-icon.png` - Automation process image
- `jobs-icon.png` - Job posts image
- `email-icon.png` - Email tracking image

Recommended size: 400x400px (square)

## Alternative: Use Free Stock Images
If you don't have images, you can use these free sources:
- Unsplash (https://unsplash.com)
- Pexels (https://pexels.com)
- Freepik (https://freepik.com)

## After Adding Images
The site will automatically display them. No code changes needed!
